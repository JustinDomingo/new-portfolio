MiniPost

MiniPost is a full-stack social media application that allows users to share text posts with one another. MiniPost utilizes React and its hooks to handle the client side and to also send asynchronous requests to the API. The application also performs CRUD operations through the text posts created by the users.

MiniPost API

This is the backend API for the MiniPost application. All documentation will be on the README page of the github repository but the backend basically validates users through Express and Node and pushes them into a MongoDB database. It also grants a JWT with a cookie after a successful register/login. Along with this, the API handles users' posts which are validated and pushed into a MongoDB database.
